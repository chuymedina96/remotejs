Remote.js

Remote Control for Web Apps

Created by Hage Yaapa / @hacksparrow

What is Remote.js?

A JavaScript library to remotely control web apps using audio frequencies.

How does it Work?

Map action to frequency

Transmitter: Send frequency

Receiver: Receive frequency

Receiver: Trigger action if frequency matches

Possible Uses

  • Media controls
  • Games - remote controlled, multi-player
  • Slideshows
  • Malware
  • Interesting browser extensions
  • Pranks
